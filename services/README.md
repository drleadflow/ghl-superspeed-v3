# Services

One file per service. Frontmatter is the source of truth — the body explains nuance.

## How to add

1. Copy `_template.md` to `<service-slug>.md`
2. Fill in frontmatter (name, duration, booking URL, intake)
3. Add description, booking notes, and link any campaigns that use it
