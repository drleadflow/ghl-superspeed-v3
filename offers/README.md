# Offers

One file per offer. An offer is a packaged set of services + pricing + terms.

## How to add

1. Copy `_template.md` to `<offer-slug>.md`
2. Fill in frontmatter (name, price, term, payment method)
3. List included services as wikilinks to `services/`
